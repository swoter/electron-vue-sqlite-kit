# Vue-ElementUI-Electron-Sqlite3 Starter

Vue + Electron + Element UI + SQLite starter kit / boilerplate with Babel, hot reloading, testing, linting and a working example app built in. It's a fork of vue-cli-plugin-electron-builder by @iswoter


## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn start
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
